📊 ANOVA Report – Filtration Time
📌 Description
This project analyzes the influence of three factors on filtration time:


Type of liquid (water, milk, oil)


Type of filter (thick, thin)


Volume (15 ml, 30 ml, 45 ml)



🧪 Methodology


Full factorial experimental design (3×2×3)


Three-way ANOVA


Analysis of interactions between factors


Assumption checks:


Independence


Normality of residuals


Homoscedasticity




Logarithmic data transformation


Post-hoc tests (Tukey)



📈 Results


All three factors have a significant effect on filtration time


Liquid type is the most influential factor


Significant interactions were observed


The model explains 99.8% of the variability



🛠️ Tools Used


SAS


Statistical analysis


Data visualization
